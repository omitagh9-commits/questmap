# QuestMap | 資格取得・勉強目標ロードマップ管理アプリ

資格取得や学習目標を設定し、試験範囲のロードマップを登録・管理するためのスタイリッシュなWebアプリケーションです。

## 主な機能
- **目標（クエスト）の新規登録・編集・削除**:
  目標期限や目標の説明を設定できます。
- **試験範囲ロードマップ（ネスト構造）の視覚化**:
  大項目、中項目、小項目をアコーディオンツリー形式で階層的に管理します。
- **学習記録の管理 (末端ノード)**:
  各学習項目（葉ノード）に対して、具体的な**使用教材（参考書）**、**達成度（0%〜100%）**、**最終勉強日時**を入力・表示できます。
- **「今勉強した！」クイック記録**:
  1タップで現在の時刻を最終勉強日時として自動登録します。
- **進捗の自動親子集計**:
  末端項目の達成度を更新すると、上階層のカテゴリおよび目標全体の合計進捗率が自動的に再計算されます。
- **ブラウザへのデータ自動永続化**:
  `localStorage` を使用しており、ページをリロードしてもデータは維持されます。
- **プリセットテンプレートの搭載**:
  「統計検定準一級」および「基本情報技術者試験」の試験範囲ロードマップがプリセットされています。

## 技術スタック
- HTML5 (セマンティックマークアップ)
- Vanilla CSS (CSS変数、Glassmorphism、レスポンシブデザイン)
- Vanilla JavaScript (再帰ツリー処理、localStorageデータバインディング)

## 動かし方
サーバーの設定等は不要です。

1. 本フォルダの `index.html` をお好みのブラウザ（Google Chrome, Microsoft Edge, Safari等）でダブルクリックして開くだけで動作します。
2. ローカルサーバーを起動して開くことも可能です：
   ```bash
   # Pythonの場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx live-server
   ```
   ブラウザで `http://localhost:8000` もしくは指定されたアドレスにアクセスしてください。
