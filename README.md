# QuestMap | 資格取得・勉強目標ロードマップ管理

「QuestMap」は、資格試験や学習目標を設定し、試験範囲を入れ子状のロードマップとして可視化・管理できるスタイリッシュな学習目標管理Webアプリケーションです。

---

## 🌟 主な機能

### 1. 目標（クエスト）の登録と管理
- 挑戦したい資格試験や学習目標（例: 「統計検定準一級取得」など）を設定。
- 目標達成の期日やメモを登録してダッシュボードで一覧管理できます。

### 2. 試験範囲の入れ子形式ロードマップ（自在なカスタマイズ）
- 試験範囲や学習ロードマップを、大項目 ＞ 中項目 ＞ 詳細知識のように階層的に管理できます。
- 画面上で直感的に「分野の追加」「項目の名前変更」「削除」を自由に行うことができます。

### 3. 学習状況の個別管理
- 各学習項目に対して、使用している**「教材（参考書名）」**、現在の**「達成度（0%〜100%）」**、**「最終勉強日時」**を記録できます。

### 4. 学習タイマー（ストップウォッチ）機能
- 学習を始める際に「再生」ボタンを押すことでタイマーが起動し、勉強時間を計測。
- 終了時に、今回の勉強時間がその項目の「累計学習時間」として自動加算され、学習日時が現在時刻に自動アップデートされます。

### 5. 進捗率と学習時間の自動親子集計
- 末端の項目を更新すると、親カテゴリーや目標全体の「合計進捗率」と「総学習時間」が自動で再計算され、プログレスバーや円形チャート、サマリーにリアルタイム反映されます。

---

## 🔒 データの安全性について

本アプリはクラウドデータベースを使用していますが、**データはお客様（ログインした本人）のアカウントにのみ紐付いて安全に保管されます。**
他人にあなたの勉強時間やロードマップ、メールアドレスが公開されたり共有されたりすることはありません。複数のデバイス（PC、スマホ）から同じアカウントでログインして、同じデータを同期して使用できます。

---

## 🚀 使い方
1. 公開サイトのアドレスにアクセスします。
2. **「新規登録」** タブから、メールアドレスとパスワードを設定してアカウントを作成します。
3. ログイン完了後、ダッシュボードから目標を設定し、ロードマップに沿って学習を開始しましょう！
