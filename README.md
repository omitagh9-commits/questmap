# QuestMap | 資格取得・勉強目標ロードマップ管理アプリ

資格取得や学習目標を設定し、試験範囲のロードマップを登録・管理するためのスタイリッシュなWebアプリケーションです。

本アップデートにより、**Firebase** を利用した**メールログイン機能**と**Firestoreクラウドデータベース保存機能**が追加され、不特定多数のユーザーが複数の端末（PC、スマホ）からデータを共有して利用できるようになりました。

---

## 主な機能
- **Firebaseメール/パスワード認証**: 
  アカウントを作成し、自分専用のダッシュボードにログインできます。他人にデータが公開されることはありません。
- **目標（クエスト）の登録・編集・削除**:
  目標期限や目標の説明を設定できます。
- **試験範囲ロードマップ（ネスト構造）の視覚化・編集**:
  大項目、中項目、小項目をアコーディオンツリー形式で階層的に管理。さらにインラインで「子要素の追加」「項目の削除」「項目の名前変更」が自在に行えます。
- **学習記録の管理 (末端ノード)**:
  各学習項目（葉ノード）に対して、具体的な**使用教材（参考書）**、**達成度（0%〜100%）**、**最終勉強日時**を入力・表示できます。
- **学習タイマー（ストップウォッチ）機能**:
  項目ごとに「勉強を開始（再生ボタン）」を押し、学習時間をカウント。終了時にその項目の「累計学習時間」として加算され、最終勉強日時が現在時刻に自動アップデートされます。
- **進捗と学習時間の自動親子集計**:
  末端の達成度や勉強時間を更新すると、上階層のカテゴリおよび目標全体の合計進捗率と総学習時間が自動的に再計算されます。
- **クラウドへのリアルタイム永続化**:
  操作はすべて Firebase Cloud Firestore と自動同期され、ページをリロードしてもデータは完全に維持されます。
- **プリセットテンプレート**:
  「統計検定準一級」および「基本情報技術者試験」の試験範囲ロードマップがプリセットされています。

## 🛠️ Firebase の接続・初期設定手順

アプリを起動してクラウド保存を有効にするには、ご自身の Firebase 接続キーを記述する必要があります。

### 1. Firebase プロジェクトの作成
1. [Firebase Console](https://console.firebase.google.com/) にアクセスし、Googleアカウントでログインします。
2. 「**プロジェクトを追加**」をクリックし、適当なプロジェクト名（例: `questmap-app`）を入力してプロジェクトを作成します（Google アナリティクスはオフにして構いません）。

### 2. 認証（Authentication）の設定
1. 左メニューの「**構築**」 > 「**Authentication**」を選択し、「**始める**」をクリックします。
2. サインイン プロバイダの一覧から「**メール / パスワード**」を選択し、ステータスを「**有効**」にして保存します。

### 3. データベース（Cloud Firestore）の設定
1. 左メニューの「**構築**」 > 「**Firestore Database**」を選択し、「**データベースの作成**」をクリックします。
2. 以下の項目を設定します：
   - ロケーション: 近い地域を選択します（例: `asia-northeast1` (東京)）。
   - ルール: 最初は開発しやすいため「**テストモードで開始**」を選択します。（本番運用する際は適切なセキュリティルールを設定してください）。
3. 「**作成**」をクリックします。

### 4. 設定ファイルの書き換え
1. 作成した Firebase プロジェクトのトップページで、「**ウェブ (</> アイコン)**」をクリックしてアプリを登録します。
2. 画面に表示される `firebaseConfig` オブジェクトの設定値をコピーします：
   ```javascript
   const firebaseConfig = {
     apiKey: "...",
     authDomain: "...",
     projectId: "...",
     storageBucket: "...",
     messagingSenderId: "...",
     appId: "..."
   };
   ```
3. 本プロジェクトの [firebaseConfig.js](file:///c:/Users/arata/OneDrive/アプリ開発/day1/firebaseConfig.js) をテキストエディタで開き、プレースホルダー部分をコピーした設定値に書き換えて保存します。

1. [firebaseConfig.js](file:///c:/Users/arata/OneDrive/アプリ開発/day1/firebaseConfig.js) の設定が完了したら、`index.html` をブラウザでダブルクリックして開きます。
2. アカウント登録およびログインが成功すると、ダッシュボードが開き、アプリが正常に利用できるようになります。
